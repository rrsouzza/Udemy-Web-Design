Lógica de Programação - Mais Operadores 2

== -> equivalência, comparação

2 == 2 -> retorna um boolean. No caso, true, pois 2 é igual/equivalente a 2.

2 != 2 -> retorna um boolean. No caso, false, pois 2 não é diferente de 2.

'2' == 2 -> retorna um boolean. No caso, true, pois é comparado apenas o valor de cada lado, e não seus tipos ('texto' == numero).

'2' === 2 -> retorna um boolean. No caso, false, pois '2' (texto) é diferente de 2(número). === compara equivalência tanto do valor quanto o tipo.

'3' === 2 -> retorna um boolean. No caso, false, pois '3' (texto) é diferente de 2(número). === compara equivalência tanto do valor quanto o tipo.

'3' === '3' -> retorna um boolean. No caso, true, pois '3' (texto) é igual a(texto). === compara equivalência tanto do valor quanto o tipo.

'3' !== 2 -> retorna um boolean. No caso, true, pois '3' (texto) é diferente de 2(número). !== compara diferença tanto do valor quanto o tipo.
