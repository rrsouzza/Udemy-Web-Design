Mais sobre semântica

  • <kbd> : keyboard; indica uma tecla do teclado (nota: também alterou o visual do texto abaixo)

  • <abbr> : abbreviation; indica uma abreviação. Carrega o atributo title, que pode ser usado para descrever o título completo da abreviação. Por padrão insere uma sublinha na abreviação, mas é possível formatá-la no CSS.

  • <time> : time; indica uma data ou horário

    Podemos definir o atributo datetime para declarar qual a data que está sendo inserida. Devemos sempre passar no padrão (YYYY-MM-DD).
